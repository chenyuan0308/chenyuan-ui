插槽 ​

组件使用 Element Plus 的 el-button 作为基础，因此支持 Element Plus 按钮组件的所有属性。

CSS 变量 ​

组件提供了以下 CSS 变量，可以用来自定义样式：

css
.common-btn-style {
  --button-height: 56px;
  --button-padding: 15px 20px;
  --button-font-size: 14px;
}
